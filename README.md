# 习概随机抽题测试

这是一个用于《习近平新时代中国特色社会主义思想概论》复习的静态网页小工具。网站基于整理后的重点章节题库制作，可随机抽取一道题进行背诵检测，并在点击后显示对应答案。

## 在线访问

GitHub Pages 地址：

```text
https://zr-can.github.io/xigai-quiz/
```

如果页面更新后没有立即变化，可以使用带版本参数的链接强制刷新，例如：

```text
https://zr-can.github.io/xigai-quiz/?v=latest
```

## 功能说明

- 随机抽取一道复习题；
- 点击“显示答案”后展示完整参考答案；
- 答案会标明所属章节与题号；
- 支持“下一道题”随机切换；
- 支持章节筛选；
- 支持本轮不重复抽题；
- 支持重置本轮抽题记录；
- 支持复制当前题目答案；
- 支持快捷键：
  - `Space`：显示 / 隐藏答案；
  - `N`：下一道题。

## 项目结构

```text
xigai-quiz/
├── index.html   # 网站主页面，包含样式、交互脚本和题库数据
└── README.md    # 项目说明文档
```

当前版本将左右背景素材以内嵌 Base64 的形式写入 `index.html`，因此不需要额外上传图片文件。

## 本地使用

直接下载 `index.html`，双击后用浏览器打开即可离线使用。

## 部署说明

本项目使用 GitHub Pages 部署：

1. 进入仓库 `Settings`；
2. 打开 `Pages`；
3. Source 选择 `Deploy from a branch`；
4. Branch 选择 `main`；
5. 目录选择 `/root`；
6. 保存后等待 GitHub Pages 发布。

## 更新说明

如需修改题库内容，可直接编辑 `index.html` 中的 `const QUESTIONS` 数组；如需修改界面样式，可编辑 `<style>` 标签中的 CSS。
